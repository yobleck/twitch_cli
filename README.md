# twitch_cli
browse twitch, watch streams, interact with chat

## Requires
  - python 3.7+
  - ANSI escape character capable terminal
  - mpv video player


## Controls
  - "q" to quit
  - "w" and "s" to scroll channel selection
  - "p" to start playing stream with mpv
  - "c" to join or leave chat
